# Linear and Non-Linear Formulas

This is a simple C++ program that displays **linear and non-linear mathematical formulas** based on user input.

## Features

- Shows linear formulas for one or multiple variables.
- Shows non-linear formulas including:
  - Quadratic formula
  - Exponential formula
  - Trigonometric formula
- Uses a **class** and **switch-case** statements.

## How to Run

1. Clone the repository or download the source code.
2. Compile the program using a C++ compiler.
